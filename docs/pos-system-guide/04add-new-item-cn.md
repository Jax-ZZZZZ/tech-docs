#### 4️⃣ 如何添加新的菜品
**Add New Item in POS**

- 点击主屏幕 **「后台管理」**
- 点击 **「菜品」**
- 在对应组别中点开 **空白菜品格子**
- 输入 **菜品名称**
- 选择对应的 **厨房打印机**
- 点击 **「保存」**
- 在价格界面点击 **「新增价格」**
- 选择 **「默认价格」** 并输入价格
- 点击 **「保存价格」**
