#### PAX S80 基本操作说明

> 说明：PAX S80 已停产，仅供维护使用。  
> S80 为按键式终端（非触屏），主要通过 **FUNC + 方向键 + ENTER** 进行操作。

---

#### 1. Sale（付款）

用于正常收款交易。

**操作步骤：**

1. 在空闲界面直接输入 **交易金额**
2. 按 **ENTER**
3. 默认进入 **Sale / Credit** 交易
4. 按屏幕提示让客人刷卡 / 插卡 / 手动输入卡号
5. 等待交易处理完成
6. 终端打印交易小票

#### 2. Void Sale（取消付款）

用于**当天交易且尚未进行 Batch（结算）之前**的取消操作。

**操作步骤：**

1. 在空闲界面按 **FUNC**
2. 使用方向键选择 **Void / V.Sale**
3. 按 **ENTER**
4. 输入要取消的 **交易编号（Trans # / Invoice #）**
5. 按 **ENTER** 确认
6. 系统处理完成并打印 Void 小票

> ⚠️ 已完成 Batch 的交易无法 Void，只能使用 Refund。

#### 3. Add Tips（添加小费 / Tip Adjust）

用于**交易已完成但尚未 Batch** 的订单补录小费。

**操作步骤：**

1. 在空闲界面按 **FUNC**
2. 使用方向键选择 **Tip Menu**
3. 按 **ENTER**
4. 输入对应的 **交易编号**
5. 输入小费金额
6. 按 **ENTER** 确认并打印调整后的小票

> 常用于客人签字后再补录小费的情况。

#### 4. Batch（结账 / 结算 / Batch Close）

用于将当天所有已完成交易**提交给银行进行结算**。

**操作步骤：**

1. 在空闲界面按 **FUNC**
2. 使用方向键选择 **Batch**
3. 按 **ENTER**
4. 选择 **Batch Close / Settlement**
5. 按 **ENTER** 确认
6. 等待系统处理完成并打印 Batch 报表

> ⚠️ 建议每天营业结束后至少进行一次 Batch。

#### 5. Force Auth（强制授权 / Force）

用于在**已获得银行授权码（Auth Code）**的情况下完成交易，  
常见于网络异常或语音授权后补录交易。

**操作步骤：**

1. 在空闲界面按 **FUNC**
2. 使用方向键选择 **Force / Force Auth**
3. 按 **ENTER**
4. 输入交易金额
5. 按提示刷卡 / 插卡 / 手动输入卡号
6. 输入银行提供的 **Auth Code**
7. 按 **ENTER** 完成交易并打印小票

> ⚠️ 此功能风险较高，仅在明确已获得银行授权码时使用。

#### 6. Refund（退款 / Return）

用于**已完成 Batch 的交易退款**。

**操作步骤：**

1. 在空闲界面按 **FUNC**
2. 使用方向键选择 **Return / Refund**
3. 按 **ENTER**
4. 输入退款金额
5. 按提示刷卡 / 插卡 / 手动输入卡号
6. 按 **ENTER** 确认
7. 系统处理完成并打印退款小票

> 退款到账时间通常为 **3–7 个工作日**，具体以客人银行为准。
