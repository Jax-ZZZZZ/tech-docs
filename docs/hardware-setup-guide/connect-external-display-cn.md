#### 第二显示屏（9.7 / 12.1 / 15 英寸）及 CDP 安装说明

<img
  src="/images/cdp.png"
  alt="2nd display"
  class="img-md"
/>

**步骤 1：**  
使用螺丝将第二显示屏固定在显示器上。

**步骤 2：**  
将显示屏连接线插入 **VGA 接口**。

**步骤 3：**  
请联系 **Beyond Menu 技术支持**，由技术人员指导进入 BIOS 设置并开启第二显示屏输出。

> **说明：**  
> 使用 **CDP** 时，**无需进行 BIOS 设置**。

---

#### 支持的显示设备
- CDP
- 9.7 英寸显示屏
- 12.1 英寸显示屏
- 15 英寸显示屏
