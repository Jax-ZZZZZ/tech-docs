#### Caller ID Modem 连接说明

Caller ID Modem 用于在来电时，将电话号码传输到 POS 系统中进行识别。

<img
  src="/images/callerid.png"
  alt="caller id setup"
  class="img-md"
/>

---

#### 连接步骤

**步骤 1：连接电话线（Telephone）**  
使用电话线，将 **Caller ID Modem 的 Phone 接口** 连接到 **电话机（Telephone）**。

**步骤 2：连接墙面电话线（Line）**  
使用电话线，将 **Caller ID Modem 的 Line 接口** 连接到 **墙上的电话接口**。

**步骤 3：连接电脑（USB）**  
将 **Caller ID Modem 的 USB 线** 插入 POS 电脑的 **USB 接口**。

---

#### 连接完成说明
- Caller ID Modem 会位于 **电话机与墙面电话线之间**
- 所有来电会先经过 Caller ID Modem，再传输到 POS 系统
- 连接完成后，请联系 **Beyond Menu 技术支持** 在 POS 软件中进行设置与测试
