#### POS 启用前必读

在正式开始使用 POS 系统前，请务必核实以下关键信息是否准确无误：

- 餐馆名称（Restaurant Name）
- 餐馆地址（Business Address）
- 餐馆联系电话（Business Phone Number）
- 餐馆适用税率（Sales Tax Rate）
- 餐馆银行收款账户（Bank Deposit Account）

⚠️ **重要提示：**  
以上信息将直接影响以下关键环节：

- 小票与账单显示
- 每日报表 / 月报表
- 交易款项结算与入账
- 税务合规
- 后续技术支持与客服处理

请务必在 **POS 正式启用前** 完成全部信息核实，  
以避免结算延误、账务错误或后续调整带来的风险。
