
### 餐馆初次使用POS必核实以下信息
- 餐馆名称
- 餐馆地址
- 餐馆电话
- 餐馆税率

### 刷卡机初次使用，请务必了解以下事项
1. 余额不足$1000左右的借记卡(debit card)，对于餐馆这种类型的商户，银行会预先收取客人20%的小费金额，防止借记卡超额付款的情况。
例如： 餐馆在刷卡机上输入的是刷$100,但是在客人的手机或者短信通知的时候，会显示被收了$120，这个就是由于上述的原因。
当餐馆晚上进行Batch操作 后,客人银行里的金额会恢复正常金额。

2. 推荐支付方式顺序为：Tap/Contactless/ApplePay(无接触式付款) > Chip(芯片卡) > Swipe(刷卡) > Manual(手动输入), 
对应的安全程度由高 到低，支付IC费用相对应的也是由低到高(Tap和Chip一样)。
其中Chip之前的方式，客人是无法通过报盗卡行为(fradudulent case)来进行拒付 (chargeback)。 
这并不表示芯片卡之前的这些支付方式就一定没有拒付，对于不同类型的拒付，要提供不同的资料。

3. 对于手动输入的方式，虽然输入卡号，过期日期即可通过刷卡，但是我们默认设置强制要求商户输入安全码(CVV)来增加安全。
然而，对于手动输入卡 号的方式，银行默认是偏向于保护持卡人，所以如果客人报拒付，必须满足以下4点：
客人签字，安全码匹配，邮编地址匹配(zipcode)，地址号码( Billing Address Number)匹配，才有可能赢回拒付金额。
对于大金额的订单，不论是芯片卡，尤其是手动输入卡号方式的订单，务必让客人出示 信用卡，最好也有驾照(Driver License),以防止盗卡行为，因为一般来说，盗卡行为，客人是无法获得实体卡片的。

4. 因为Visa会定期进行抽查审核，如果发现餐厅有不当附加费、现金折扣和/或设定最低交易金额作为销售条件相关的重大违规行为， Visa是有权力餐厅进行高额罚款的。
如果餐馆想收取客人的信用卡服务费，一定要注意落实以下几点：
    - 借记卡没有最低交易限额。
    - 信用卡的最低交易金额最多可设置为 10 美元。
    - 现金折扣是指以信用卡利率显示价格（如菜单上所示）。现金支付还可享受额外折扣。但是，显示现金价格并增加信用卡付款费用将被视为附加费，是不允许的。
    - 附加费不适用于借记卡。
    - 入口处和销售点必须张贴标牌。附上所需通知的示例。
    - 信用卡附加费可以是固定费用，也可以是一定百分比，但不能超过交易金额的 3%。
    - 马萨诸塞州、康涅狄格州、缅因州和俄克拉荷马州禁止收取附加费。
    - 仅当使用第三方平台付款时才可能收取便利费，例如 BeyondMenu Online、UberEat 或 Doordash。
    - 仅允许向政府机构、医院和学校等公共机构收取服务费。
